# Ecotourism Nodes

[Español](es/README.md)

An open and replicable model for activating ecotourism circuits through local coordination, lodging, and progressive infrastructure.

## One-line summary

Ecotourism Nodes is an open framework for seeding place-based tourism circuits that combine reliable lodging, local experiences, and gradual infrastructure improvements in ways that can later become more autonomous and locally sustained.

## Why this exists

Many places with high natural, cultural, and scenic value do not capture more tourism because they lack a workable combination of:

- reliable lodging
- clearly articulated local activities and services
- minimum support infrastructure
- a simple interface for visitors who do not know the area, local payment methods, local contacts, or the local language well

In places like Mallasa, there is also a strong weekly imbalance: weekends bring many local visitors, while tourism drops sharply during the week.

## What an ecotourism node is

An ecotourism node is a territorial circuit that starts by connecting what already exists and progressively strengthens it through:

- lodging with a reliable minimum standard of comfort
- coordination of local activities and providers
- simple visitor-facing communication, booking, and support
- gradual infrastructure improvements
- public documentation so the model can be replicated elsewhere

## Core principles

- The circuit does not replace local offerings; it makes them more visible, coordinated, and sellable.
- It starts with what already exists and improves infrastructure and coordination progressively.
- It integrates diverse actors without making any single actor indispensable.
- The value created should stay increasingly within the area.
- Learning, structure, and improvements are documented publicly to support replication.
- The model is open to multiple lodging formats, not just one.

## Lodging as an enabler

The initial pilot may start with an RV because it can provide a minimum standard of comfort where that standard may not yet exist.

An RV is not the end goal. It is only an initial enabler.

Over time, a node may also incorporate:

- camping
- hosted stays in local homes
- other mobile units
- simple local lodging formats
- future fixed accommodations

## Seed operator role

At the beginning, one person or team may act as the seed operator, covering two functions:

### Coordination
- presenting the offer to visitors
- coordinating activities and providers
- managing payments, bookings, and follow-up
- handling communication and friction points

### Deployment
- moving, installing, and removing lodging units
- cleaning and logistical support
- water, drainage, and basic operational tasks
- maintaining a minimum quality standard

This role should become progressively more distributable over time.

## What remains in the area

The goal is for the value created by each node to remain increasingly in the territory, including:

- useful infrastructure
- improved visibility and reputation
- local know-how and documented processes
- more articulated demand for local actors
- future local lodging capacity
- stronger local operating autonomy

## Growth path

A node can grow in stages:

1. start with a minimum viable circuit using existing assets
2. validate demand, pricing, and local coordination
3. improve logistics and support infrastructure
4. integrate more lodging formats and providers
5. move toward greater local autonomy
6. replicate the model in other territories

## Public documentation and replicability

Ecotourism Nodes is meant to be documented publicly in a simple and reusable way so that both the lessons and the structure can help other places build similar circuits.
